# Infinite Scroll — Website

Marketing site and download hub for [Infinite Scroll](https://github.com/judegao/infinite-scroll), built with Next.js.

## Stack

- Next.js 16
- React 19
- Tailwind CSS 4
- TypeScript

## Development

```
npm install
npm run dev
```

Open [localhost:3000](http://localhost:3000).

## Deploy

```
npm run build
```
